# Narrative Arcs

## The Six Basic Story Shapes (Kurt Vonnegut)

1. Man in hole (down → up)
2. Boy meets girl (up → down → up)
3. From bad to worse (down → down)
4. Cinderella (up → down → up higher)
5. Icarus (up → down)
6. Oedipus (down → up → down)
