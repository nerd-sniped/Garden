# Plot Structure

## Common Structures

- **Three Act Structure**: Setup → Confrontation → Resolution
- **Hero's Journey**: Ordinary world → Call → Return
- **Freytag's Pyramid**: Exposition → Rising → Climax → Falling → Resolution
- **Save the Cat**: beat sheet (15 beats)
