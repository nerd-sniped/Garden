# Arrays and Lists

- **Array**: contiguous memory, O(1) index access, O(n) insert/delete
- **Linked List**: O(1) insert/delete at head, O(n) index access
- **Dynamic Array** (Vec, ArrayList): amortised O(1) append

See [[Data Structures Overview]] · [[Algorithms Overview]].
