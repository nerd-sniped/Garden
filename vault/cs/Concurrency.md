# Concurrency

## Concurrency Primitives
- **Threads** — shared memory, requires locks
- **Async/Await** — cooperative multitasking
- **Channels** — message passing (Go, Rust)
- **Actors** — isolated state (Elixir, Akka)

See [[Go]] · [[Rust]] · [[Elixir]] · [[Memory Management]] · [[CS Concepts Hub]].
