# Memory Management

## Strategies
- **Manual** (C, C++) — full control, easy to leak
- **GC** (Java, Python, Go) — automatic, pause overhead
- **Ownership** (Rust) — compile-time safety, zero cost
- **ARC** (Swift) — reference counting

See [[Rust]] · [[C Plus Plus]] · [[Go]] · [[CS Concepts Hub]].
