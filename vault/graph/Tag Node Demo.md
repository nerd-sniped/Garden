# Tag Node Demo

**Tag nodes** are octahedron-shaped nodes automatically added to the graph for every unique tag.

Clicking a tag node in the 3D graph **dims all unrelated notes** and highlights only the notes connected to that tag. A filter banner appears — click ✕ to clear.

This note demonstrates several tags:

```yaml
tags: [graph/features, graph/tags]
```
