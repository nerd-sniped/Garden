# Databases Overview

| Type | Examples | Best for |
|---|---|---|
| Relational (SQL) | PostgreSQL, MySQL, SQLite | Structured data, transactions |
| Document | MongoDB, CouchDB | Flexible schema |
| Key-Value | Redis, DynamoDB | Cache, sessions |
| Graph | Neo4j, ArangoDB | Highly connected data |
| Column | Cassandra, BigTable | Time-series, analytics |
| Search | Elasticsearch | Full-text search |
