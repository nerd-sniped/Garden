# Testing Strategies

## Testing Pyramid
- Unit tests (many, fast)
- Integration tests (some)
- E2E tests (few, slow)

## Types
- **Unit**: isolated function tests
- **Integration**: multi-component tests
- **E2E**: browser-level automation (Playwright, Cypress)
- **Snapshot**: UI regression
