# Note Taking Principles

- Write in your own words
- Be atomic: one idea per note
- Connect to existing notes immediately
- Titles should be complete statements
- Never copy/paste without processing

See [[Evergreen Notes]] · [[Zettelkasten Method]] · [[Productivity Hub]].
