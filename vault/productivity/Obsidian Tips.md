# Obsidian Tips

Tips for using Obsidian as a PKM tool.

- Use `[[wikilinks]]` to connect ideas — they form the graph
- Tags cluster related notes visually
- Use the graph view to discover unlinked notes
- Obsidian Git plugin for auto-sync
