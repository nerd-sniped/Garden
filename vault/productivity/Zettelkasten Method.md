# Zettelkasten Method

Zettelkasten is a note-taking system by Niklas Luhmann. Each note is an atomic idea, linked to others.

**Three note types:**
1. Fleeting notes — quick captures
2. Literature notes — from sources
3. Permanent notes — processed ideas
