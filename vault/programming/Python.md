# Python

Python is a high-level, dynamically-typed language famous for its readability.

## Key Features
- Dynamic typing
- First-class functions
- Comprehensive stdlib
- Async/await

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("World"))
```
