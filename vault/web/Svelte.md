# Svelte

Svelte compiles components to vanilla JS at build time — no virtual DOM.

```svelte
<script>
  let name = "World";
</script>
<h1>Hello, {name}!</h1>
```

See [[Vue]] · [[React]] · [[Web Technologies Hub]].
