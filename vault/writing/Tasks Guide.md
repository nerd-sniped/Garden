# Tasks Guide

Use GFM task list syntax:

```markdown
- [ ] Unchecked task
- [x] Completed task
- [ ] Another pending task
```

Renders as:

- [x] Deploy the site
- [x] Add all graph features
- [ ] Add 200+ notes
- [ ] Connect to Netlify
- [ ] Remove template notes

See [[Writing Hub]].
